  • 数据库操作
    • 查询所有数据库
    • 创建数据库
    • 删除数据库
  • 表操作
  • 文档操作
  • 操作符

本节主要介绍 MongoDB 的数据库操作。

# 查询所有数据库

show dbs;
1

或者：

show databases;
1

# 创建数据库

use <database_name>;
1

如果数据库不存在，则创建数据库，否则切换到指定的数据库。

# 删除数据库

先执行 use <database_name> 命令，然后执行如下命令：

db.dropDatabase();
1

就会将当前数据库删除。
