---
title: 聚合函数
---

常用的聚合函数主要有以下几种：

- `AVG` 平均值
- `COUNT` 总数
- `MAX` 最大值
- `MIN` 最小值
- `SUM` 总和


聚合函数通常可以与 `GROUP BY` 语句进行联用，譬如：

```sql
SELECT AVG(salary)
FROM table_name
GROUP BY job;
```

上述语句，会检索出每一种 `job` 对应的平均 `salary`。

在此之下，我们还可以与 `HAVING` 子句连用，筛选出对应条件的数据：

```sql
SELECT AVG(salary)
FROM table_name
GROUP BY job
HAVING job = 'engineer' OR job = 'designer';
```

`HAVING` 的使用语法类似于 `WHERE` 语句，唯一的区别是 `HAVING` 只能用在 `GROUP BY` 条件下。
