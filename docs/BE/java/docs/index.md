`SDKMAN!`

```shell
curl -s "https://get.sdkman.io" | bash
```

自动注册环境变量，初始化：

```shell
source "$HOME/.sdkman/bin/sdkman-init.sh"
```

查看电脑可用的 Java 版本：

```shell
sdk list java
```

安装 Java：

```shell
sdk install java 17.0.12-oracle
```
