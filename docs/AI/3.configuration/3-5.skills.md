## 含义

Skills 是一组可复用的本地指令与资源，通常以 `SKILL.md` 作为入口文件，配合脚本、模板或参考资料，帮助 Agent 在特定任务上更快、更一致地完成工作。你可以把它理解为“带说明书的微型工具箱”。

## 安装

常见安装方式如下，任选其一即可：
- 手动安装：将技能仓库或文件夹放到 `~/.codex/skills/<skill-name>/`（或你的工具约定目录），确保包含 `SKILL.md`。
- 技能管理器：如果你的 Agent 提供了 `skill-installer` 之类的技能管理能力，可以通过命令或对话安装。
- 组织内共享：在团队仓库中维护统一技能集，按需同步到本地。
- 命令行（`npx skills`）：部分环境提供 `skills` CLI，可用 `npx skills` 进行技能的安装或管理，具体以 `npx skills --help` 的输出为准。

## 用法

- 触发方式：在对话里直接提及技能名（例如 `$skill-name` 或 `skill-name`）。
- 执行流程（常见约定）：先读取 `SKILL.md` 按步骤执行；如 `SKILL.md` 指向脚本或模板，优先使用这些资源；避免一次性加载大量无关文件，按需读取即可。

## 优秀示例及介绍

以下示例仓库可作为参考，建议从它们的 `SKILL.md` 结构与工作流设计中学习：
- [ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)：专注 UI/UX 设计到实现的流程化指导。
- [frontend-design](https://github.com/anthropics/claude-code/tree/main/plugins/frontend-design/skills/frontend-design)：前端设计与视觉实现的成套规范与示例。
- [skill-creator](https://github.com/anthropics/skills/blob/main/skills/skill-creator)：指导如何创建高质量技能的官方示例。
- [notebooklm-skill](https://github.com/PleasePrompto/notebooklm-skill)：将外部知识库接入 Agent 的实践范例。
- [ralph-wiggum](https://github.com/anthropics/claude-code/blob/main/plugins/ralph-wiggum)：轻量、有趣的技能示例，适合学习结构与触发方式。
- [vue-skills](https://github.com/antfu/skills)：Vue 生态相关技能集合，涵盖框架与最佳实践。
- [react-best-practices](https://github.com/vercel-labs/agent-skills/tree/main/skills/react-best-practices)：React 项目实践与规范的技能化整理。
